# Smart Port AI

Projet de simulation d’un système d’intelligence artificielle pour prédire le temps d’attente des camions dans un port.

Technologies :
- Python
- Pandas
- Scikit-learn
- Streamlit

Fonctionnalités :
- Prédiction du temps d’attente
- Interface utilisateur
- Machine Learning

## Lancer l'application
streamlit run appSGTD.py

## Objectif
Prédiction du temps d'attente des camions dans un port.
